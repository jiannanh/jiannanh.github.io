计算机网络 - 网络层

2020-08-09

IP
主要任务是把分组从源端传到目的端，为分组交换网上的不同主机提供通信服务。网络层传输单位是数据报。
功能1：路由选择与分组转发最佳路径
功能2：实现异构网络的互联
功能3：拥塞控制

数据交换方式

多路复用

报文交换

优点：

  1. 无需建立连接
  2. 存储转发动态分配线路
  3. 线路可靠性高
  4. 线路利用率较高
  5. 多目标服务

缺点：

  1. 有存储时延
  2. 报文大小不确定，需要网络罗节点偶较大的存储空间。

分组交换

  1. 无需建立连接
  2. 存储转发，动态分贝线路
  3. 线路可靠性较高
  4. 线路利用率较高
  5. 有利与报文交换，存储管理容易

缺点：

  1. 有存储时延
  2. 需要额外的信息量
  3. 乱序到目的主机时，要对分组排序重组

数据报

无连接：不是先为分组的传输确定传输路径，每个分组独立确定传输路径，不同分组传输路径可能不同。
